Directives in Angular-

Directives are classes that add additional behavior to elements in your Angular applications.

Here are some of the most common Angular directives:

1.ngFor: This directive is used to iterate over a collection of data and render a list of elements.
2.ngIf: This directive is used to conditionally render an element based on a condition.
3. ngSwitch: This directive is used to render different elements based on a value.
4. ngStyle: This directive is used to set the CSS styles of an element based on dynamic data.
5. ngClass: This directive is used to add or remove CSS classes from an element based on dynamic data.

Types of Directives-


Attribute Directive

They are used to change the appearance or behavior of an element.

For example - ngClass, ngStyle

Structural Directive

They are used to change the DOM layout.

For example - ngFor, ngIf, ngSwitch

Component Directive

They are used to create reusable components that can be added to other components.

For example - Highlighter, Dropdown


ROUTING AND IT'S TYPES

Routing in Angular is a mechanism that allows users to navigate between different views of an application.
It is a core feature of Angular that enables the creation of Single Page Applications.

There are three types of routing in Angular:

Component-based routing:

This is the most common type of routing and is used to navigate between different components. When a user clicks on a link, the router navigates to the corresponding component and displays its view.

Navigation-based routing:

This type of routing is used to navigate between different views of the same component. When a user clicks on an item in the list, the router navigates to the details view of that item.

Data-driven routing:

This type of routing is used to navigate based on the data in the application.
